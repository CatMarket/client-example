# CatMarket API client

API doc: https://api.catmarket.io/api-docs/

## Install

```bash
npm install
```

## Configure

```
cp .env_example .env
```

set the PRIV_KEY and MARKET_KEY in .env

## Run

```bash
ts-node src/client.ts makebuy 45ee725c2c5993b3e4d308842d87e973bf1951f5f7a804b21e4dd964ecd12d6b_0
```
